MIMI'S VILLAIN ORIGIN STORY - TXT BATCH PACK
Generated from the uploaded Mimi character/game list, the 150 origin idea table, and the game plan.

Core analysis:
- Mimi should stay silly, sincere, clumsy, emotionally dramatic, and self-aware rather than cruel or genuinely dark.
- The best loop from the plan is: start screen -> choose category or fate -> 3 mystery cards -> reveal inconvenience -> choose reaction -> villain meter -> villain result -> play again.
- The 150 origin ideas work best as a hidden content database. Players should see only CATEGORY + TEASER before clicking, then the real inconvenience and villain story reveal after the choice.
- Reaction choices should be randomized from separate calm/chaos batches so the game feels interactive without needing complex logic.
- Extra randomized batches included here: category cards, villain meter text, transition lines, result labels, continue/play-again buttons, powers, weaknesses, catchphrases, and calm outcomes.

Suggested simple JS structure per origin:
{
  id,
  category,
  cardTitle,
  teaser,
  inconvenience,
  villainName,
  normalImage,
  villainImage,
  story,
  power,
  weakness,
  catchphrase,
  calmOutcome
}
